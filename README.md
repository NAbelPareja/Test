
# 🧠 Quiz App

Aplicación web de preguntas tipo quiz desarrollada con Angular.

## 🚀 Funcionalidades

- Preguntas dinámicas desde archivo JSON
- Selección de respuestas
- Temporizador por pregunta
- Evaluación automática
- Resultados con respuestas correctas e incorrectas

## 🛠 Tecnologías

- Angular
- TypeScript
- HTML / CSS

## ▶️ Cómo ejecutar el proyecto

1. Clonar el repositorio
```bash
git clone https://github.com/NAbelPareja/Test.git
```
2. Entrara a la carpeta
```bash
cd Test
```

3. Instalar dependencias
```bash
npm install
```

4. Ejecutar proyecto
```bash
ng serve
```
